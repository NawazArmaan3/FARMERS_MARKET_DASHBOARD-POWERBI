# FARMERS_MARKET_DASHBOARD-POWERBI


This repository contains an interactive Farmers Market Dashboard built in Microsoft Power BI Desktop. The dashboard provides a data-driven view of sales performance, customer behavior, vendor contributions, and product demand across the farmers market.

Designed with clean visuals, slicers, and dynamic filtering, the dashboard helps stakeholders understand key market trends and make informed decisions.

⸻

 Project Overview

The Farmers Market Power BI Dashboard transforms raw market transaction data into meaningful insights using:
 • Data modeling
 • DAX measures
 • Interactive visuals
 • Dynamic slicers
 • Custom formatting

The dashboard highlights:
 •  Total Sales / Total Quantity Sold
 •  Total Customers
 •  Unique Customers
 •  Total Vendors
 •  Total Booths
 •  Product Sales & Quantity
 •  Vendor Sales Performance
 •  Customer ZIP Code Sales
 •  Customer-Level Performance
 •  Word Cloud of Top Customers

⸻

 Key Dashboard Features

1. High-Level KPI Metrics

Displayed across the top of the dashboard:
 • Total Sales / Total Quantity
 • Total Customers
 • Total Vendors
 • Total Booths
 • Unique Customers

These metrics instantly show overall market performance.

⸻

2. Product Name vs Sales / Quantity

Power BI pie and bar charts used to analyze:
 • Which product categories drive the most revenue
 • How quantity sold differs across product types

⸻

3. Vendor vs Sales / Quantity

A donut chart showing:
 • Vendor contribution percentage
 • Total sales per vendor
 • Comparative vendor performance

⸻

4. Customer vs Sales / Quantity

A bar visualization that highlights:
 • Top performing customers
 • Quantity purchased per customer
 • Customer buying behavior

⸻

5. Customer ZIP vs Sales / Quantity

A ZIP-based chart showing:
 • Geographic trends
 • ZIP codes contributing the most revenue
 • Regional purchasing patterns

⸻

6. Customer Name Word Cloud

A Power BI visual that:
 • Emphasizes key customers
 • Shows frequency based on sales contribution

The larger the name, the more valuable the customer.

⸻

7. Interactive Slicers

Users can dynamically filter the entire dashboard by:
 • Customer ID
 • Vendor
 • Product
 • Customer ZIP

All visuals update instantly using Power BI’s filtering engine.
